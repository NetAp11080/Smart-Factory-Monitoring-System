# Smart-Factory-Monitoring-System
This project is based on Industrial Automation using IoT, where factory processes are monitored and controlled automatically using sensors and microcontrollers. It helps improve efficiency, reduce human effort, and enable real-time monitoring.
# Features
📡 Real-time monitoring
🤖 Automation of machines
🌐 Remote access using IoT
⚡ Energy efficient system
# Components
ESP32 / Arduino
Temperature Sensor
IR Sensor
Relay Module
Power Supply
# Working
Sensors collect data
ESP32 processes data
Relay controls devices
Data sent to cloud/app
